# 🛡️ ThreatGuard  
### Detect Scam Emails & Malicious URLs using Flask + AI

ThreatGuard is a cybersecurity web application that helps detect:

- 📧 Scam / Fake Emails
- 🔗 Phishing & Malicious URLs

Built using Python, Flask, and AI-inspired phishing detection logic.

---

# 🚀 Features

✅ Detect phishing and scam URLs  
✅ Analyze uploaded TXT and PDF files  
✅ Scam email/message detection  
✅ User-friendly web interface  
✅ Real-time detection results  
✅ Cybersecurity-focused project  

---

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/ThreatGuard.git
```

---

## 2️⃣ Open Project Folder

```bash
cd ThreatGuard
```

---

## 3️⃣ Create Virtual Environment

### Windows

```bash
python -m venv venv
```

Activate virtual environment:

```bash
venv\Scripts\activate
```

---

## 4️⃣ Install Dependencies

```bash
pip install flask PyPDF2 python-dotenv
```

---

# ▶️ Run The Project

```bash
python main.py
```

Open browser:

```bash
http://127.0.0.1:5000
```
